# Jianghu Dynasty v2.7.0 - Wulin Congress and Martial Crisis

- Added Wulin gathering decisions and event flavor.
- Added public speech, mediation, traitor accusation, martial alliance, and recognition decisions.
- Added lightweight event-based Wulin tournament outcomes.
- Added martial crisis response events for manual theft, frontier alarms, sect scandal, and court pressure.
- Added non-title recognition such as Wulin mediator, protector, alliance speaker, respected hero, disputed figure, and suspected opportunist.

The system is event-based and cooldown-limited, with no global scans.
