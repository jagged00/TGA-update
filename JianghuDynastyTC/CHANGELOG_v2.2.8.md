# Jianghu Dynasty v2.2.8 - Jianghu Gameplay Core

- Added fresh additive vanilla-map gameplay files for modifiers, decisions, events, and localization.
- Added optional Jianghu reputation modifiers: low, respected, renowned, and legendary.
- Added martial path decisions for training, challenges, master visits, recovery, and weapon forms.
- Added military path decisions for service, field assignment, troop training, officer networking, and campaign study.
- Added court and formal service decisions without depending on a custom Chinese title.
- Added a small optional event pool for training, duels, masters, military discipline, court service, and reputation consequences.

All systems are optional. No XP, levels, class locks, fixed endings, or mandatory quest chains were added.
