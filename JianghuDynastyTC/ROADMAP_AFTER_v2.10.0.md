# Roadmap After v2.15.0

## v2.15.x Stable Hotfixes Only

- Runtime error log cleanup.
- Localization corrections.
- Balance tuning.
- Decision visibility fixes.
- Save/load compatibility notes.

## Frozen Vanilla Map Feature Scope

The v2.x vanilla-map edition should not receive large new systems after v2.15.0. It should remain stable and upload-ready.

## v3.0.0 Experimental

Custom Song map work belongs only in a separate experimental branch/workspace. Do not merge custom map files into v2.x stable.
